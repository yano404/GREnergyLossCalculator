GR Energy Loss Calculator
=========================

## Requirements

- numpy
- [pycatima](https://github.com/hrosiak/pycatima)

## Usage

```
./calc_eloss.py -b <magnetic field(mT)>
./calc_eloss.py --mag <magnetic field(mT)>
```

or

```
./calc_eloss.py -n <run number>
./calc_eloss.py --run <run number>
```
